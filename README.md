# apnabazaar
PakBazaar Marketplace

PakDeal ek simple aur modern marketplace website frontend hai jahan users cars, bikes, mobiles, houses, electronics aur dusre items ki listings dekh sakte hain. Yeh project HTML, CSS aur JavaScript se banaya gaya hai.

Features

* Responsive homepage design
* Search bar for filtering listings
* Category section for quick browsing
* Modern listing cards with image, title, price, and location
* Contact seller button
* Clean and sleek UI design

Technologies Used

* HTML5
* CSS3
* JavaScript

Project Structure

marketplace/
├── index.html
├── style.css
└── script.js

How to Run

1. Project folder download ya clone karein.
2. Folder ko apne computer par open karein.
3. index.html file ko browser mein open karein.
4. Marketplace frontend website run ho jayegi.

Search Functionality

Search bar listing cards ke titles ko filter karta hai. Agar search text listing title se match kare, to woh card display hoga; warna hide ho jayega.
